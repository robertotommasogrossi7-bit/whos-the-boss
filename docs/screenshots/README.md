# Screenshots

Metti qui i PNG e poi **togli il commento** alle righe immagine nel README principale
(`README.md` / `README.it.md`).

Scatti consigliati (tieni nomi semplici e coerenti):

| File | Cosa mostrare |
|---|---|
| `home.png` | Home "Segna partita" con la GameBar in alto. |
| `standings.png` | Classifica: lega + globale "La tua situazione" (prima riga aggregata). |
| `match-log.png` | Lo sheet "Esito partita" (vincitori / pareggio / partecipanti). |
| `poker-table.png` | Il tavolo poker (tema feltro). |
| `method.png` | Uno scatto di `METODO.md` o della history dei commit (il processo). |
| `chat.png` | Uno spezzone di una chat di orchestrazione (facoltativo). |

Suggerimento: schermate **mobile-width** (~390px) rendono meglio per un'app mobile-first.
